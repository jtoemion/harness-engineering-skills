# Football365 - Navigation Recipes

## football-news
Navigate to homepage and extract latest article links.

```
1. [goto] https://www.football365.co.uk/
2. [wait] 3 seconds
3. [extract] article links
```

## headlines
Get the main headline stories from the homepage.

```
1. [goto] https://www.football365.co.uk/
2. [wait] 3 seconds
3. [extract] headline titles
```

## team-news
Navigate to a specific team's news section.

```
1. [goto] https://www.football365.co.uk/teams/{team-name}
2. [wait] 3 seconds
3. [extract] article links
```
