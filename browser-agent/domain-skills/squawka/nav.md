# Squawka Navigation Recipes

## Home & Search
- **Homepage**: Navigate to `https://www.squawka.com`
- **Search**: Use the search icon in the top navigation bar. Enter team name, player, or competition.

## Live Scores
- **Live Scores Page**: `https://www.squawka.com/en/live-scores`
- **Today's Matches**: Available on homepage by default

## Teams & Players
- **Team Pages**: Search for team name → select from autocomplete
- **Player Profiles**: Search for player name → select profile

## Competitions
- **Premier League**: `https://www.squawka.com/premier-league`
- **La Liga**: `https://www.squawka.com/la-liga`
- **Champions League**: `https://www.squawka.com/champions-league`
- **Other competitions**: Use navigation menu or search

## Match Pages
- **Match Centre**: Available via team page or live scores → click specific match
- **Statistics**: Tabbed sections within match pages (Stats, Lineups, Commentary, Odds)

## Common Workflows
1. **Find specific match**: Home → Live Scores → click match → Match Centre
2. **Compare teams**: Search team → Team page → Stats/History
3. **Player stats**: Search player → Player Profile → Statistics tab
