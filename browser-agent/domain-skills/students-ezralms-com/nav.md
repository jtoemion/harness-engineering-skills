## Troubleshooting

### Stagehand API Key Missing
Error: "OpenAI API key is missing. Pass it using the 'apiKey' parameter or the OPENAI_API_KEY environment variable."
Fix: Use playwright_browser_* tools (Playwright) instead of stagehand_* for browser automation.

### LMS Network Never Idles
EzraLMS uses Firebase long-polling which never triggers networkidle.
Fix: Use [wait] for specific elements or text instead of waiting for networkidle.

### Lesson HTML CSS Overrides
The lesson viewer applies its own CSS that overrides custom styles.
Fix: Use inline styles only (style="..."). CSS classes in custom HTML are ignored in /lesson/{id}.

### Slow Page Loads
Some pages (Students, Tasks) take 5+ seconds to fully render.
Fix: Use [wait] with explicit text/element conditions and longer timeouts.

### Snapshot UID Refresh
UIDs change on every navigation/page load.
Fix: Always call take_snapshot before interacting after navigation.

### Modal/Overlay Blocking
Some modals may block clicks on underlying elements.
Fix: Use fill_form or click with force:true if clicks don't register.

## Selector Priority (Most Reliable)

1. URL navigation - goto /class/{id}
2. Role selectors - button "Add Topic", textbox "Username or Email"
3. Heading text - heading "Learning AI"
4. Link text - link "Classes"

Avoid: Snapshot UIDs (transient), partial text matching (unreliable)
