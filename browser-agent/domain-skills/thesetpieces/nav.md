# The Set Pieces - Navigation Recipes

## latest-news
Navigate to homepage and extract latest article links.

```
1. [goto] https://www.thesetpieces.com/
2. [wait] 3 seconds
3. [extract] article links
```

## headlines
Get the main headline stories from the homepage.

```
1. [goto] https://www.thesetpieces.com/
2. [wait] 3 seconds
3. [extract] headline titles
```
